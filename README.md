# chrono-musician-code-adventure
out of home chrono code musician security on rails adventure
- premiere samourai , ecrivain, ou pirate etait 1 femme
- on écrire dans un cli (command line) comme si c'est une bataille navale en temps reeel
- la security est un login sur rails
- on doit aller hors des sentis battus sur les reseaux sociaux (sans faire de reseaux social illegal)
- ton arme de destruction est la MUSIQUE
- des affiches out of home t'aident à diffuser la musique sur les reseaux
🎵 Chrono Musician Code Adventure — Résumé du README
🚀 Concept général

Une aventure hybride entre code, musique, sécurité, et narration interactive, où tu incarnes un personnage hors norme dans un univers qui mélange CLI, infiltration créative et exploration des réseaux.
🧙‍♀️ Choisis ton identité

Tu peux incarner :

    un samouraï,

    un écrivain,

    ou un pirate,
    …et dans cette histoire, le premier héros est une héroïne.

⌨️ Le gameplay

    L’expérience se déroule dans un CLI, comme une bataille navale en temps réel.

    Chaque commande est une action, un mouvement, une attaque ou une création.

🔐 La sécurité

    Le système repose sur un login Rails, simple mais symbolique :
    la sécurité fait partie de l’aventure.

🌐 Hors des sentiers battus

    Le joueur doit diffuser sa musique sur les réseaux sociaux,
    sans utiliser les chemins classiques,
    sans enfreindre aucune règle,
    mais en étant créatif, malin, et un peu rebelle.

🎶 Ton arme : la MUSIQUE

    La musique est ton outil, ton pouvoir, ton moyen d’expression et de conquête.

    Elle sert à débloquer des zones, influencer le monde, et toucher les autres joueurs.

🪧 Out-of-home

    Des affiches physiques (OOH) dans la ville deviennent des relais :
    elles t’aident à diffuser ta musique dans le monde numérique.

    créer une version plus narrative,

    ou proposer une structure professionnelle pour GitHu
